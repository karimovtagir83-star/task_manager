# Task Tracker

Консольное приложение для управления задачами на Python.

Проект позволяет создавать, просматривать, удалять задачи и отмечать их как выполненные. Все задачи сохраняются в JSON-файл и восстанавливаются после перезапуска программы.

## Возможности

- Добавление новых задач
- Просмотр списка задач
- Удаление одной задачи или всех задач
- Отметка одной задачи или всех задач как выполненных
- Сохранение задач между запусками программы
- Загрузка сохранённых задач из JSON
- Обработка отсутствующего или повреждённого JSON-файла

## Структура проекта

```

TaskTracker/
│
├── main.py        # Запуск программы, меню и обработка команд
├── manager.py     # Логика работы с задачами и сохранение данных
├── task.py        # Класс Task
└── tasks.json     # Файл с сохранёнными задачами

````

## Используемые технологии

- Python 3
- JSON
- ООП (классы и объекты)
- Работа с файлами
- Модули Python

## Установка

Клонировать репозиторий:

```bash
git clone <repository_url>
````

Перейти в папку проекта:

```bash
cd TaskTracker
```

Запустить программу:

```bash
python main.py
```

## Использование

```
Choose language (rus/eng)
```

## Команды

### Добавление задачи

Команда:

```
add <название> <описание>
```

Пример:

```
add Python Учить JSON
```

Создаёт новую задачу.

---

### Просмотр задач

Команда:

```
list
```

Показывает все существующие задачи и их статус.

Пример:

```
Index: 1
Task: Python
Description: Учить JSON
Status: Pending
```

---

### Выполнение задачи

Команда:

```
complete <название>
```

Пример:

```
complete Python
```

Также можно выполнить все задачи:

```
complete all
```

---

### Удаление задачи

Команда:

```
delete <название>
```

Пример:

```
delete Python
```

Удалить все задачи:

```
delete all
```

---

### Выход

Команда:

```
exit
```

Перед выходом программа сохраняет все изменения.

## Как работает сохранение

При запуске:

```
tasks.json
      ↓
json.load()
      ↓
объекты Task
```

При сохранении:

```
объекты Task
      ↓
словарь Python
      ↓
json.dump()
      ↓
tasks.json
```
